Во второй части строку подключения (private static string connect) нужно будет изменить на свою. Её можно найти:

вид -> обозреватель объектов sql server -> кликнуть на нужный сервер правой кнопкой мыши, свойства (если там нет нужного сервера, то его надо добавить) -> с самого верху общие свойства, третья снизу надпись -- строка подключения

P.S. Если не работает, проверяем:

1) служба сервера запущена (Sql server configuration manager)

2) в скопипасченой строке подключения знак \ заменён на \\

3) нужная база данных создана, т.е. часть 1 выполнилась успешно

P.P.S. Мы создаём базу в tempdb2 так как tempdb -- это уже существующая папка и её лучше не трогать
